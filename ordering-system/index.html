Chow Town

Welcome to our project!

This project was a work in progress over the course of a few weeks. We worked on this project as the final project for COMP 3020 at the University of Manitoba.

My role was to learn the tech stack, distribute workload among group members based on their skill levels, and edit/design the HTML/CSS/JavaScript structure.

My group consisted of:

  • Andy Kim - Data Flow (Query strings between pages, checkout page)
  • Abu Yasin Sabik - Dashboard Structure (Dashboard page structure and styling)
  • Melvin Philemon - Dashboard Structure (Dashboard page, basic styling, page linking)
  • Yalun Lyu - Additional Information (Header link information)
  • All - General layout design, ideation, and basic prototyping/testing.
